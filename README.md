# Daily Quest

Version: 0.1.0 (Work in Progress)

A personal challenge and habit tracking application built with Python and Tkinter.

This project is designed to track long-term goals, daily habits, and personal challenges.

---

## Current Features

* Tkinter graphical user interface
* Push-ups challenge tracker
* Add daily push-ups progress
* Input validation
* Error handling for invalid values
* Progress display updates in real time

---

## Planned Features

* Automatic day counter
* Multiple challenges
